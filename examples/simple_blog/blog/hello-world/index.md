# Hello World

Welcome to my very first blog post! This blog was built using **Blogatto**, a static site generator for Gleam.

## What is Blogatto?

Blogatto is a framework for building static blogs with Lustre and Markdown. You write your content in Markdown, define your templates with Lustre, and Blogatto takes care of the rest:

- Parsing markdown with frontmatter
- Rendering HTML via customizable components
- Generating RSS feeds
- Creating sitemaps
- Producing robots.txt
